# This is pulse! A game I made for shiba
**Key Features:**
 - Parkour
 - has elements of puzzles
 - really cool graphics
 - custom menu and home screen
 - made on godot
 - has 2 player mode(not made till now)
 - Really satisfying! click sound when you hover on stuff and it zooms in!
